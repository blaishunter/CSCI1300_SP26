# Hello Blanks Warm-Up

## Overview
This is a short activity to review the basic structure of a C++ script. Students are expected to complete the script by filling in the blanks.

## Instructions
1. Using the provided HelloBlanks.cpp file...
2. Fill in the blanks using the word bank
3. Not all words in the word bank will be used
4. Run your program. This should print "Hello World!" twice

## Commands
Compile: 

g++ -Wall -Werror -Wpendantic -std=c++17 HelloBlanks.cpp

Run / Execute:

Windows/Linux: ./a.exe
Mac: ./a.out
